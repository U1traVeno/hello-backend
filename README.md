# Hello Backend

你好，未来的后端开发者！

本项目是 2025 HDU-CS-Wiki 共学营的后端附加任务项目。

这个项目也许会是你的第一个后端项目！`hello-backend`是一个为你精心搭建的后端入门模板，旨在帮助你迈出从零到一的后端学习关键一步！

不用担心，这个项目是冲着新手友好去的。你不是一个人在战斗！
